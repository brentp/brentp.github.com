Correlation

I see a lot of publications doing correlations on methylation data. It's something that I've thought about a lot as well.

A particular cytosine may be always methylated. If we measure that cytosine with 2 technologies, say Illumina methylation chip, and then by pyrosequencing we want to see a high correlation--indicating that the pyrosequencing has validated the data from the chip.

If all that's reported is the correlation, we may misunderstand what's actually occuring.

To illustrate this, I plot some random points below at a site that's measured as 95 to 100 % methylated by one techology (x-axis) and 90 to 95% methylated in another.

def point_plot(xs, ys):
    plot(xs, ys, 'bo')
    xlim(0, 1)
    ylim(0, 1)
    title("r = %.3f" %  np.corrcoef(xs, ys)[0, 1])

import numpy as np
np.random.seed(42)
 
# generate random data that's tightly clustered near 1.0
xs = np.random.randint(950, 1000, size=20) / 1000.
ys = np.random.randint(900, 950, size=20) / 1000.
 
point_plot(xs, ys)

Note that, by one interpretation, the above plot indicates that both technologies are saying the same thing -- that all samples are completely methylated.

However, if all we report is the correlation, we see that there's a correlation of -0.211. That's bad!

Below, the addition of a single sample (data otherwise unchanged from above) drastically changes that correlation.

In [32]:
 
xs = [0.1] + list(xs)
ys = [0.2] + list(ys)
 
point_plot(xs, ys)

Note that in the above plot, the addition of a single point changes the correlation from -0.12 to 0.992 even though that sample has the largest disparity between the 2 axes (0.1 :: 0.2).

Of course, this is nothing particularly deep. It makes sense, however, it seems to be overlooked quite often.
